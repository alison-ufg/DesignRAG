# Replication Package: DesignRAG

Welcome to the anonymous replication package for the paper submitted for peer review: **"Improving the Quality of AI-Generated User Interfaces through Multimodal Retrieval of Proprietary Design System Artifacts"**.

⚠️ **Note on Intellectual Property:** The source code of the DesignRAG *backend* inference engine and the Figma integration microservices developed in this research are subject to Intellectual Property (IP) agreements and industrial secrecy, and cannot be made publicly available. 
However, in strict compliance with Open Science guidelines for empirical evaluations, this repository provides **all raw data, the reference set (*ground truth*), the complete inferences of all evaluated models, and the statistical *scripts***, ensuring full auditability and reproducibility of the study's quantitative analysis.

---

## 📁 Repository Structure

The repository is logically organized to facilitate a step-by-step audit of the experiment:

* `/dataset/`
  * `miniatures/`: Contains the 207 pixel visual renderings (PNG) used in the research.
  * `data_json_context.json`: Extracted metadata (JSON), structural properties, and the semantic context generated by multimodal LLMs.
  * `prompt.md/`: Prompt used to generate the context.
* `/queries/`
  * `queries.json`: Dataset containing the 30 stratified search intents (in English and Portuguese) and the validated official ground truth mapping.
* `/demo/`
  * Anonymized video demonstration of the pipeline execution in real-time.
* `/stats/`
  * `Analysis_P@K_MRR/` *(Descriptive Analysis)*
    * `metrics.json`: Raw performance results (P@k and MRR) for all queries across all models.
    * `Analysis.py`: Script to extract retrieval averages per model.
    * `results_metrics.txt`: **[Output] Consolidated table with averages (P@1, P@3, P@5, MRR)**.
  * `Fleiss_Kappa/` *(Ground Truth Validation)*
    * `team_responses.csv`: Raw annotations from the expert panel (individual selections).
    * `Fleiss_Kappa.py`: Script to calculate the level of agreement among evaluators.
    * `result_Fleiss_Kappa.txt`: **[Output] Fleiss' Kappa agreement score**.
    * `consolidated_ground_truth.py`: Script that processes majority voting and generates the ground truth.
    * `result_consolidated_ground_truth.csv`: **[Output] Official ground truth used in the experiment**.
  * `Friedman_Wilcoxon/` *(Inferential Statistics)*
    * `mrr_results.json`: Isolated MRR result vectors for the 30 queries.
    * `Friedman_Wilcoxon.py`: Script to calculate the Global Test (Friedman) and pairwise comparisons (Wilcoxon *signed-rank* with Holm-Bonferroni correction).
    * `result_Friedman_Wilcoxon.txt`: **[Output] Statistical significance results**.
  
---

## 📊 Summary of Statistical Results

The effectiveness of the DesignRAG architecture was validated through non-parametric tests applied to the retrieval scores (MRR). The detailed results (available in the `/stats` folder) attest that:

1. **Expert Agreement:** The *Ground Truth* validation achieved a Fleiss' Kappa score of **0.2630** (Moderate Agreement), establishing a solid baseline for evaluating the natural ambiguity of UI components.
2. **Global Difference:** The Friedman Test confirmed a highly significant global variation in the performance of the evaluated models (**$p < 0.0001$**).
3. **Multimodal Superiority:** Pairwise Wilcoxon tests proved that all multimodal variants (DesignRAG) statistically outperformed both unimodal baselines (FTS and Purely Textual RAG), with **$p_{adj} < 0.05$**.

## 🚀 How to Reproduce the Statistical Analysis

Evaluators can audit the math and p-values reported in the paper by running the provided Python scripts. 

**Step-by-step for execution:**
1. Click the **Download (ZIP)** button in the top right corner of this page to download the complete replication package.
2. Unzip the file on your local machine and open the terminal in the extracted folder.

**Prerequisites:**
* Python 3.10+
* Libraries: `numpy`, `packaging`, `pandas`, `patsy`, `python-dateutil`, `scipy`, `six`, `statsmodels`

**Terminal Execution:**
```bash
# Install analysis dependencies
pip install -r stats/requirements.txt

# 1. Consolidate the ground truth and validate the expert panel:
python stats/Fleiss_Kappa/consolidated_ground_truth.py
python stats/Fleiss_Kappa/Fleiss_Kappa.py

# 2. Calculate descriptive metric averages (Precision and MRR):
python stats/Analysis_P@K_MRR/Analysis.py

# 3. Audit statistical significance (Friedman and Wilcoxon):
python stats/Friedman_Wilcoxon/Friedman_Wilcoxon.py
